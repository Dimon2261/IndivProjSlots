# IndivProjSlots
This is repo for my slots simulator project

В данном репозитории вы можете увидеть мою реализацию симулятора слот машины с дополнительными функциями
Я пишу код в VisualStudio и ориентируюсь только под Windows, на других ОС и в других редакторах код может не запускаться без внесения изменений в проект либо отображаться некорректно

Из функций на данный момент реализовано:
-сама слот машина
-возможность просмотра истории спинов
-возможность сбросить сохранение и началь с начала
-возможность сохранить прогресс и выйти

Все сохранения хранятся в файле save.txt в формате:
1 строка - текущий баланс
Последующие n*2 строк (где n - кол-во спинов, совершенных на данном сейве)  - изменения баланса, где отрицательным числом обозначается ставка, а положительным - выигрыш

Касательно самих слотов. Из стратегий выигрыша реализованы линии (lines), группы символов (clusters) и символы джекпота (scatter)
Линия - несколько одинаковых символов (3 или больше), выстроенных в прямую линию
Группа - несколько одинаковых символов (3 или больше), каждый из которых соприкасается с идентичным себе символом, образуя непрерывную фигуру
Джекпот - 3 или больше символа S на барабане в любых местах

Диагонали в текущей версии проекта не учитываются ни в одной стратегии выигрыша. В планах добавить нахождение диагональных и ломанных линий и отредактировать коэффициенты выигрыша под них
